Inicializar repositorio local
git init
Ver estado de archivos
git status
Agregar archivos al Stage
git add archivo
Agregar modificaciones al repositorio
git commit -m "comentario"
Historial de commit
git log
Historial de commit abreviado
git log --oneline

Copiar Repo-local a --> Repo-remoto
git remote add origin [url:repositorio-remoto]
git branch -M main
git push -u origin main
Actualizar Repo-local a --> Repo-remoto
git push
Copiar Repo-remoto a --> Repo-local
git clone [url:repositorio-remoto]
Actualizar Repo-remoto a --> Repo-local
git pull
Mostrar vinculos remotos
git remote -v

Listar archivos
ls
Listar archivos incluyendo los Ocultos
ls -a
Ver donde estoy
pwd
Cambiar a otro directorio
cd [namefolder]
Regresar un directorio atras
cd ..
Regresar al directorio raiz
cd ~
Crear directorio
mkdir [namefolder]
Remover directorio
rmdir [namefolder]
Crear archivo
touch [file.ext]
Remover archivo
rm [file.ext]
Mover o Renombrar
mv [file_from][file_to]

Verificar si existe un SSH
ls -al ~/.ssh
Generar las Keys SSH -- opcion 1
ssh-keygen -t rsa -b 4096 -C "tu-correo"
Generar las Keys SSH -- opcion 2
ssh-keygen -t ed25519 -C "tu-correo"
Correr agente SSH en Gitbash
eval "(ssh-agent -s)"
Agregar key Privada en Gitbash
ssh-add ~/.ssh/id_rsa
Copiar contenido key Publica para usar en Github
clip < ~/.ssh/id_rsa.pub

Configuracion de Usuario
git config --global user.name "tu-nombre"
Configuracion de correo
git config --global user.email tu-correo
Configuracion de Editor (vscode)
git config --global core.editor "code --wait"
Configuracion de Salto de linea (windows)
git config --global core.autocrlf true
Mostrar configuracion
git config --list
